# CT34 Apps — kho add-on Home Assistant riêng

Kho này chứa 1 add-on: **Chấm công & lương - CT34** (thư mục `chamcong_ct34/`).

## Cách cài vào Home Assistant

1. Home Assistant → **Settings → Apps → App Store**.
2. Bấm **⋮** (góc trên phải) → **Repositories**.
3. Dán: `https://github.com/tnthanhlan/chamcongCT34` → **Add**.
4. Đóng lại, kéo xuống App Store sẽ thấy mục **"CT34 Apps"** mới xuất hiện, bên trong có app **"Chấm công & lương - CT34"** → **Install**.

Chi tiết cấu hình (mật khẩu, Cloudflare Tunnel...) xem file `chamcong_ct34/README.md`.
